## When to Use These Docs

- **Before writing code**: Check `code-standards.md` for formatting and type requirements
- **Understanding the system**: Review `architecture.md` for component relationships
- **Implementing tool calls**: Reference `mcp-tools-quick-ref.md` for exact parameters
- **Working on agent behavior**: Consult `agent-behavior.md` for expression patterns
- **Running tests/builds**: Use `dev-commands.md` for quick command reference
